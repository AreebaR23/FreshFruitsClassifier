# Experiment Results

This directory stores evaluation results, plots, and analysis.

## Contents

- **Confusion matrices**: `{model}_confusion_matrix.png`
- **Training history plots**: `{model}_training_history.png`
- **Evaluation metrics**: `{model}_results.pth`
- **Model comparisons**: `model_comparison.png`

## Metrics Saved

For each model, the following metrics are saved:
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- Per-class performance
- Misclassified samples analysis
